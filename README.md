# Student Attendance Dashboard During COVID-19

## Project Overview
This project analyzes student attendance trends during the COVID-19 pandemic using PostgreSQL, SQL, Excel, and Power BI.

The dashboard explores:
- Attendance status (Present, Late, Absent)
- Learning mode comparisons (Remote, Hybrid, In-Person)
- Attendance by grade level
- Attendance trends over time
- Student attendance counts and KPI metrics

The dataset was created and managed in PostgreSQL using SQL queries and then connected to Power BI for visualization and dashboard reporting.

---

## Tools Used
- PostgreSQL
- SQL
- DataGrip
- Power BI
- Excel

---

## Key Insights
- Remote learning was the most common learning mode during the analyzed period.
- Attendance behavior varied across different grade levels.
- Grade 9 showed the highest combination of attendance events.
- Attendance status distribution highlighted differences between Present, Late, and Absent records.
